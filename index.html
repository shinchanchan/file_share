Help & Guide

How It Works

Rocket Share enables direct file transfers between devices using WebRTC technology. No files are uploaded to servers.

Quick Start

  • Share your ID with the receiver
  • Enter receiver's ID and select files
  • Tap "Launch Transfer" to begin
  • Receiver gets download notification
  • Track progress in real-time

Connection Types

Direct: Devices connect directly when on same network

TURN Relay: Uses relay server when direct connection fails

Tips

  • Keep app open during transfers
  • Use stable internet connection
  • Monitor connection type indicator
  • Check downloads by tapping the download icon
